# PowerBI_Dashboard_Project-Netflix
# 📺 Netflix Stock Prediction Dashboard
# 🌟 Overview
This project features an interactive Power BI Dashboard designed to analyze and visualize Netflix's historical stock market performance. 📊 The dashboard provides a high-level view of price trends, volatility, and trading volume, helping users understand the stock's behavior over time.

By integrating various financial metrics, the system transforms complex datasets into an intuitive, interactive experience for investors or data enthusiasts. 💡

# ✨ Features
Interactive Time-Series Tracking: Dynamic line charts that allow users to filter by year, quarter, or month to see specific price movements. 📅

Key Performance Indicators (KPIs): Instant visibility into crucial metrics like Average Closing Price, High/Low Peaks, and Total Volume Traded. 🔑

Trend Identification: Visual representations of price fluctuations to spot bullish or bearish patterns. 📈

Volume Analysis: Integrated charts showing the relationship between trading activity and price changes. 🔊

Slicer-Based Navigation: Easy-to-use filters for custom date ranges and specific financial attributes. 🎛️

# 🛠️ Technologies Used
Power BI Desktop: For data modeling, DAX calculations, and report authoring. 🖥️

Power Query: To clean, transform, and shape the stock market dataset. 🧹

DAX (Data Analysis Expressions): Used to create custom measures for financial calculations. 🔢

# 📊 Key Functionalities
Comparative Analysis: Compare opening vs. closing prices across different fiscal periods.

Volatility Mapping: Identifies the spread between daily highs and lows to visualize market risk.
